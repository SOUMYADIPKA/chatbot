# chatbot
AI Voice Assistant

An advanced AI-powered voice assistant built using Python that combines speech recognition, natural language processing, and system automation — inspired by JARVIS.

🚀 Features
🎤 Voice & Speech
Speech Recognition using Whisper
Multi-language support (English, Hindi, Bengali)
Text-to-Speech (TTS) with natural voice (Zira)
Wake word activation ("Hey Jarvis")
Voice retry mechanism for accuracy
🧠 AI & NLP
Natural Language Processing using spaCy
Intelligent command understanding
Context-aware responses (basic)
Self-learning capability using SQLite database
💻 System Automation
Open/close applications (Calculator, Notepad, etc.)
System controls (shutdown, restart)
Media playback control
Time & basic information responses
🌐 Internet Integration
Google search for unknown queries
AI-powered responses using OpenAI API
Smart comparison between search results
🖥️ User Interface
Tkinter-based GUI
Real-time chat display
Voice + text input support
Interactive control buttons
🛠️ Technical Stack
Technology	Usage
Python	Core development
Whisper	Speech recognition
pyttsx3	Text-to-speech
spaCy	NLP processing
SQLite	Database
Tkinter	GUI
OpenAI API	AI responses
